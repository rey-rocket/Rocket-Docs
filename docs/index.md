## Platform Overview

!!! tip "What makes Rocket different?"
    Rocket goes beyond traditional point systems by integrating **real-world rewards**, **behavioral science**, and **seamless LMS integration** to create lasting engagement.

### Core Features

<div class="features-showcase" markdown>

=== "Gamification Engine"
    
    **Streaks & Ranks System**
    
    - :material-fire: **Explorer** (0-3 weeks): Building momentum
    - :material-airplane: **Pilot** (4-6 weeks): Consistent engagement  
    - :material-crown: **Commander** (7+ weeks): Peak performance
    
    Each rank unlocks exclusive discounts and privileges, creating a clear progression path that motivates continued participation.

=== "Real Rewards"
    
    **3,000+ Brand Partners**
    
    - :material-shopping: Nike, Adidas, Apple, Starbucks
    - :material-food: Deliveroo, Just Eat, Costa Coffee
    - :material-home: IKEA, Argos, John Lewis
    - :material-cart: Amazon, ASOS, Tesco, ASDA
    
    Students earn points for learning activities and redeem them for real gift cards and exclusive discounts.

=== "Smart Analytics"
    
    **Data-Driven Insights**
    
    ```mermaid
    graph LR
        A[Student Activity] --> B[Points Earned]
        B --> C[Engagement Metrics]
        C --> D[Behavioral Insights]
        D --> E[Course Optimization]
    ```
    
    Track learning patterns, identify at-risk students, and optimize course design based on engagement data.

=== "Multi-Tenant Architecture"
    
    **Enterprise Ready**
    
    - :material-domain: Separate organizational data
    - :material-security: Role-based access control
    - :material-scale-balance: Scalable float management
    - :material-file-upload: Bulk user onboarding via CSV
    
    Support multiple institutions with isolated data and customized reward catalogs.
